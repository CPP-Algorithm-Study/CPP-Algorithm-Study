### ✅ 진행 방식
1. 매주 월요일까지 미리 정해둔 5문제를 풀고 코드 리뷰를 진행한다.
2. 코드 리뷰는 자유롭게 진행한다.
   - 어려웠던 부분 공유
   - 풀이에 대한 의견 제시
       - 잘했다고 생각하는 부분
       - 이렇게 하면 더 좋을 것 같다고 생각하는 부분
       - 왜 이렇게 풀었는지 궁금한 부분
       - 또 다른 풀이 방식 제시
4. 문제는 프로그래머스 Lv2~Lv3 내에서 선정한다.
